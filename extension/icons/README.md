# Extension Icons

Place the following icon PNG files in this directory:

| File | Size |
|------|------|
| `icon16.png` | 16×16 px |
| `icon48.png` | 48×48 px |
| `icon128.png` | 128×128 px |

Icons are referenced in `manifest.json` under the `"icons"` field.
These must be present before submitting the extension to the Chrome Web Store.
For local development/demo, Chrome will load the extension without icons (uses a default puzzle-piece icon).
